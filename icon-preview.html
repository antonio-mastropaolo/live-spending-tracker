Implementation notes
Static icons: use NSStatusItem with NSImage(systemSymbolName:) in .template rendering mode — macOS handles dark/light inversion automatically. Animated icons (11–34): rendered as WKWebView or NSView with Core Animation layers. Recommended size: 13–14 pt, weight medium. For gradient/colorful icons use .alwaysOriginal rendering mode. Set statusItem.length = NSStatusItem.variableLength for the pill layout.
